URXVT problem: unknown terminal type


Sometimes while working on remote systems from urxvt, you get an error 'unknown terminal type'. In this post I'm gonna show you a simple solution. 

Find a file in your system terminfo. And copy it in your remote system's user homedir. Create a dir first in remote system by issuing `mkdir -p .terminfo/r` and then copy terminfo from your system to remote system.




scp /usr/share/terminfo/r/rxvt-unicode-256color remoteUser@remoteSystem:.terminfo/r


That's it. Now login and you'll see no error this time.
